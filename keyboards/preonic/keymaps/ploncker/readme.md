# Ploncker

Converted my Ploncker Planck layout to Preonic.

* Removed non-QWERTY layouts
* Shift numbers row left one to allow hyphen and equal keys
* Remove LED controls
* Add function layer (replaces LED layer)
* Fn+numbers is F-keys
