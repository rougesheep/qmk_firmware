# Media focused keymap for the Type-9 Macropad

Volume and music controls. The middle row are F13 F14 and F15 to be assigned to shortcuts.

```
|-------+-------+-------|
| MUTE  | VOL - | VOL + |
|-------+-------+-------|
| F13   | F14   | F15   |
|-------+-------+-------|
| PREV  | PLAY  | NEXT  |
|-------+-------+-------|
```
